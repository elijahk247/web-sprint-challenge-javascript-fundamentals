Interview Questions

1. Briefly compare and contrast .forEach & .map (2-3 sentences max)

.map() is primarily used to manipulate a data set and creates a new array unlike .forEach(). .forEach() iterates through every element of a data set and is used to inspect each element of the data set.


2. Explain the difference between a callback and a higher order function.

A higher order function is a function that takes in another function as a parameter, while a callback is the function argument that the higher order function is taking in.


3. What is closure?

Closure is the concept that when a function is created, it can reach outward for context but never being able to reach in directly.


4. Describe the four rules of the 'this' keyword.

a) Global/Window binding: when in global scope, the value of ‘this’ will be the window/console object. This is done by default and usually is not the desired output.

b) Implicit Binding: when a function is called by a preceding dot, the object left of the dot gets ‘this’.

c) New Binding: whenever a constructor function is used, ‘this’ refers to the specific instance of the object that is created and returned by the constructor function.

d) Explicit Binding: Overriding what the ‘this’ keyword is referring to using the .call() or .apply() method.


5. Why do we need super() in an extended class?

Because class is just sugar syntax in JavaScript, it means that 'class' is just another way to create constructor functions. super() abstracts the syntax .call() that we have been using in constructor functions to bind 'this' to object elements.
